# Necesidades de la aplicación para SOCIOS

## 1. Proceso de alta de nuevo socio
1. Cuando una persona quiere hacerse socio, debe rellenar el formulario.
2. Al rellenar el formulario se crea una tarea para los administradores denominada "Validar nuevo socio"
3. Este tipo de tareas tendrá un botón, en el listado de tareas, de tipo Validar
4. Cuando se pulsa sobre validar, se le vanta un dialog que muestra todos los datos del posible nuevo socio y dos botones VALIDAR - DENEGAR. También hay un espacio de texto para incluir un mensaje sobre porqué se valida o deniega. 

## 2. Primeros pasos de un nuevo socio

1. Cuando el acceso de un nuevo socio es validado, este puede acceder al sistema. Para ello se le envía un correo electrónico de bienvenida con un password aleatorio y que será obligatoriamente cambiado en su primer acceso.
2. Además de modificar el password deberá, obligatoriamente, elegir si quiere pagar las cuotas por transferencia/ingreso o mediante domiciliación. En el caso de elegir el segundo tipo, debe incluir un IBAN válido.
